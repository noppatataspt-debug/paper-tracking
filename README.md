# Paper Usage Tracker

ระบบบันทึกการใช้กระดาษโดยสแกน QR code พร้อม traceability

## Tech Stack

- **Frontend**: HTML + Vanilla JS (no build step)
- **Backend**: Supabase (PostgreSQL + REST API)
- **Hosting**: Vercel

## Deploy

```bash
vercel --prod
```

## Local Dev

```bash
python3 -m http.server 8000
```

เปิด http://localhost:8000
